# InsureSmart – Intelligent Insurance Recommendation System

## Overview

InsureSmart is a Generative AI-powered insurance recommendation platform that helps users select suitable insurance plans based on their:

- Age
- Income
- Health Conditions
- Lifestyle Habits
- Insurance Preferences

The application uses Google Gemini AI for intelligent risk analysis, premium estimation, and personalized insurance recommendations.
